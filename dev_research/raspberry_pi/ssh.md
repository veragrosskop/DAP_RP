# SSHing into the pizero

To ssh into the pizero you should be able to use the following command (assuming you are on the network or connected via LAN)

```bash
ssh pizero@pizero
```

and when it prompts you for the password you also fill in 

`pizero`

It may ask you to verify the fingerprint. You can safely type yes.

Once you are in you can execute any commands!
